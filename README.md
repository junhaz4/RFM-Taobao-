# RFM-Taobao-
This project is to analyze customers' characteristics from Taobao

The raw data is obtained by web crawling from Taobao

Description of original data:

平台    : platform

店铺名称 : Store Name

买家昵称 : Customer Name

付款时间 : Payment Time

发货时间 : Delivery Time

结束时间 : Order Ending Time

订单状态 : Order Status

实付金额 : Payment

邮费	  : Postage

省份	  : Province

城市    : City  

购买数量 : Quantity
